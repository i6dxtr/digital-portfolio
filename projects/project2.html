Cell Segmentation Challenge

AI-powered human cell segmentation using multiple machine learning approaches

Python TensorFlow Computer Vision CNN SVM Naïve Bayes
Live Demo Source Code

Project Overview

Under a deadline of 5 months, worked with a team of 4 to develop a program that segments human cells and their boundaries using artificial intelligence and machine learning concepts.

This project challenged us to explore multiple approaches to cell segmentation, comparing the effectiveness of different machine learning algorithms in accurately identifying and delineating cell boundaries in microscopic images.

Technical Approaches

  • Naïve Bayes Classification
  • Convolutional Neural Networks
  • Support Vector Machine Kernels
  • Image Preprocessing Pipeline
  • Boundary Detection Algorithms
  • Performance Comparison Analysis
  • Team Collaboration & Version Control
  • Working Examples & Documentation

Key Achievements

Successfully produced working examples of several different segmentation approaches, allowing for comprehensive comparison of algorithm performance. The project demonstrated proficiency in multiple machine learning paradigms and their application to computer vision problems.

The team collaboration aspect was crucial, as we divided responsibilities across different algorithmic approaches while maintaining consistent data preprocessing and evaluation metrics.
