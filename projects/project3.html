Project Overview

In a project team of 4, fully developed, implemented, and trained a machine learning image classification model on a set of in-store product images using TensorFlow python libraries.

This project was submitted to an accessible technologies design competition, where we ultimately placed as finalists and won $4,500 for funding and research into the product. The solution addresses accessibility challenges in retail environments by providing AI-powered product identification.

Key Features

  • Real-time Image Classification
  • TensorFlow Model Training
  • Intuitive Web Interface
  • Backend Model Hosting
  • Accessibility-Focused Design
  • Product Database Integration
  • Full-Stack Architecture
  • Competition-Ready Deployment

Technical Implementation

The project involved comprehensive machine learning pipeline development, from data collection and preprocessing to model training and deployment. We created a robust web application that seamlessly connects the frontend user interface to our trained TensorFlow model.

The accessibility focus was central to our design, ensuring that the tool could be effectively used by individuals with visual impairments or other accessibility needs in retail environments.

Competition Success

Our team's innovative approach to accessible technology earned recognition in the design competition, resulting in finalist status and $4,500 in funding for further research and development. This achievement validates the practical impact and technical excellence of our solution.
